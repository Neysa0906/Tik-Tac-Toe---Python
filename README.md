# Tic Tac Toe (Python GUI with AI)

A **GUI-based Tic Tac Toe game** developed using **Python and Tkinter** as part of my **1st Semester B.Tech (CSE) Problem Solving Using Python project**.

This project supports:
- Player vs Player
- Player vs Computer (Basic AI)

## Features
- Interactive **3x3 GUI board**
- Player **X vs Computer (O)**
- Automatic **win and draw detection**
- Winning cells are **highlighted**
- **Reset / New Game** option
- Simple **rule-based AI**

---

## Technology Stack
- **Language:** Python  
- **GUI Library:** Tkinter  
- **IDE:** Visual Studio Code  

---

## How the Game Works
- Game board is represented using a **1D list of size 9**
- Players alternate turns
- Computer selects moves using:
  - Win if possible
  - Block opponent
  - Take center
  - Take corners
  - Random move
